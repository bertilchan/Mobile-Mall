## 网易严选商城（移动端）

#### 1.目录

* mall — 前端代码
* mall-server — 服务端代码

#### 2.技术栈

* 前端：Vue + Axios + Less + Vant UI
* 后端：Ｎode.js + ThinkJS + MySQL

#### 3.主要功能

+ 首页

+ 分类首页、分类商品、新品首发、人气推荐商品页面

+ 商品详情页面，包含加入购物车、收藏商品、商品评论功能

+ 搜索功能

+ 专题功能

+ 品牌功能

+ 完整的购物流程，商品的加入、编辑、删除、批量选择，收货地址的选择，下单支付

+ 会员中心（订单、收藏、足迹、收货地址、意见反馈）
